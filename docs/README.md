# DART Project Page

This folder contains a static GitHub Pages homepage for:

**DART: Difficulty-Adaptive Routing for Zero-Shot Video Temporal Grounding**

To publish it on GitHub Pages:

1. Push the repository to GitHub.
2. Open repository `Settings` -> `Pages`.
3. Set `Source` to `Deploy from a branch`.
4. Select the branch and choose `/docs` as the folder.
5. Save the setting and wait for GitHub Pages to finish deployment.

The entry point is `index.html`. All local media assets are stored in `assets/`.
